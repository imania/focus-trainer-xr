# Privacy policy draft

Focus Trainer XR does not require an account and does not intentionally collect personal information.

## Data collection

The app stores exercise settings locally in the browser or packaged PWA runtime when needed for app functionality. These settings may include selected eye mode, focus distance range, exercise pattern, and session duration.

## Health data

The app does not collect, transmit, or sell health data. The app does not diagnose, treat, cure, or correct vision conditions.

## Network

When distributed as a web-hosted PWA, the hosting provider may receive standard web server logs such as IP address, user agent, and request timestamps.
