# Store listing draft

## App name

Focus Trainer XR

## Short description

An immersive wellness trainer for visual focus shifts and eye comfort routines.

## Long description

Focus Trainer XR provides guided visual focus-shift routines in immersive VR. Follow a glowing target through step-based near/far depth changes, randomized positions, and smooth tracking patterns. Select left-eye, right-eye, both-eye, or alternating modes, then choose balanced, near-emphasis, or far-emphasis profiles.

This app is intended as a wellness and comfort routine tool. It does not diagnose, treat, cure, or correct vision conditions. Stop immediately if you experience pain, double vision, dizziness, headache, nausea, or unusual discomfort.

## Keywords

focus, visual routine, eye comfort, wellness, WebXR, Quest

## Category suggestion

Health and fitness, or Utilities, depending on Meta Dashboard options.

## Claims to avoid

- Improves eyesight
- Cures myopia
- Treats hyperopia
- Replaces glasses or medical care
- Clinically proven vision correction
